# Project 01 - Wazuh SIEM Lab Setup and Endpoint Monitoring

## Objective

Build a basic SOC homelab using Wazuh and Sysmon to collect and monitor endpoint security events.

## Lab Environment

* Wazuh Server
* Windows 10 Endpoint
* Sysmon
* VirtualBox

## Tasks Performed

1. Installed Wazuh Server
2. Connected Windows Agent
3. Installed Sysmon
4. Verified agent communication
5. Reviewed CIS Configuration Assessment

## Results

* Wazuh dashboard successfully received endpoint events
* Windows endpoint appeared as an active agent
* CIS benchmark assessment results were generated
* Security monitoring became operational
